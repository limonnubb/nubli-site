Возможности

Всё что нужно
для полного контроля

Каждая функция оптимизирована для максимальной скорости и эффективности.

NSP — Nuke & Spawn

Мгновенное удаление всех каналов и создание новых с автоматическим заполнением сообщениями. Батчевая обработка для максимальной скорости.

/nsp

Flood

Отправка 5 сообщений от лица бота в один клик. Быстро, чисто, без задержек. Идеально для зафлудить канал.

/flood

DM Mass Mail

Массовая рассылка в личные сообщения всем участникам сервера. Батчевая отправка с таймаутами для обхода лимитов.

/dm_all

Delete Channels

Параллельное удаление всех каналов на сервере. Батчи по 10 каналов одновременно для максимальной скорости очистки.

/delete_channels

Send Message

Отправка произвольного сообщения от лица бота в текущий канал. Просто и быстро для любых нужд.

/send

Magic 8Ball

Магический шар с 12 уникальными ответами. Задай вопрос — получи предсказание в стильном эмбеде.

/8ball

Список команд

Slash-команды

Все команды используют современную slash-систему Discord с автоматической синхронизацией.

/nsp
Удалить все каналы и создать новые с автозаполнением. Параметры: количество, название, текст.
Admin
/stopnsp
Экстренная остановка процесса NSP в любой момент.
Admin
/flood
Отправить 5 одинаковых сообщений от лица бота в текущий канал.
Admin
/delete_channels
Параллельное удаление всех текстовых и голосовых каналов сервера.
Admin
/dm_all
Массовая рассылка сообщения в ЛС всем участникам сервера (кроме ботов).
Admin
/dm_test
Тестовая отправка ЛС себе для проверки работоспособности.
Admin
/send
Отправить одно сообщение от лица бота в текущий канал.
Admin
/8ball
Задать вопрос магическому шару и получить случайный ответ из 12 вариантов.
